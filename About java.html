INTRODUCTION TO JAVA

Java is a high-level, object-oriented programming language developed by James Gosling at Sun Microsystems and released in 1995. It is renowned for its core principle," Write Once, Run Anywhere" (WORA), allowing compiled Java code (bytecode) to run on any platform equipped with a Java Virtual Machine (JVM),regardless of the underlying operating system. Java programming code written for one platform, like the Windows operating system, can be easily transferred to another platform, like a mobile phone OS, and vice versa without being completely rewritten. Java works on multiple platforms because when a Java program gets compiled, the compiler creates a .class bytecode file that can run on any operating system that has the Java virtual machine (JVM) installed on it. It's typically easy to install JVM on most major operating systems, including iOS, which was not always the case.

    FEATURES OF JAVA:
    1. Object-Oriented:
      Java follows the OOP paradigm, which facilitates software development and maintenance using concepts like classes, objects, inheritance, and polymorphism.
    2. Platform Independent:
      Java code is compiled into an intermediate form called bytecode, which runs on the JVM. This bytecode can be executed on any operating system (Windows, macOS, Linux, etc.) that has the JVM installed, making it platform-independent.
    3. Simple:
      Java was designed to be easy to learn and use. It removes complex features found in languages like C++, such as explicit pointers and operator overloading
    4. Secure:
      Java has built-in security features, including automatic garbage collection and bytecode verification, which help prevent memory leaks and reduce vulnerabilities.
    5. Robust:
      It is a strong language with excellent memory management (automatic garbage collection) and exception handling, which helps detect errors both at compile time and runtime.
    6. Multithreaded:
      Java supports multithreading, allowing multiple tasks to run simultaneously within a single program, leading to efficient CPU utilization and improved performance.
    7. High Performance:
      While Java is interpreted, the Just-In-Time (JIT) compiler converts bytecode into native machine code at runtime, which significantly improves the execution speed.
    8. Distributed:
      Java is designed for distributed environments, providing features like Remote Method Invocation (RMI) and support for internet protocols (HTTP, FTP), which help in creating network-based applications.
    9. Rich API and Libraries:
      Java provides a vast set of APIs (Application Programming Interfaces) and libraries for networking, database connectivity (JDBC), GUI development, and more, which simplifies coding for developers.
